# Etch-a-Sketch

This is a webpage where you can color grid squares by hovering over them with mouse cursor, change the size of the grid
and create RGB colored grid or grid with opacity.

This grid has a 960x960 resolution and uses flexbox to fit more squares inside without changing the amount of pixels
it uses. When creating a new grid the maximum size cannot be higher than 100 (100 squares horizontal and 100 vertical).
